# TIL

1. Object.seal()

   객체나 배열을 seal메서드를 통해 길이값을 고정,

   인덱스를 적어 직접값을 수정하는것은 가능하나 push,pop이 불가

   freeze 의경우 아예 모든것이 불가 

2. 선택적 Parameter 구현 잘 쓰면 유용할듯

   ```css
   function myFunc(a,b) {
     b = b || 0;
     // b will be set either to b or to 0.
   }
   
   // 또는 
   function myFunc(a, b = 0) {
      // function body
   }
   ```
